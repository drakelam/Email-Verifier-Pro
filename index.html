How to Install

Installation is very easy with our Easy Installer. You can install yourself simply in few steps without any coding knowledge. We develop our installation system to make the installation process flawless and easy.

Installation Steps:

  1. There are 2 setups 32bit and 64bit
  2. You choose the right one for your computer.
  3. Enjoy.

In this step, I will guide everyone to use and install quickly.

First turn at the main screen interface of the software.













You choose Mail to check, here there are 3 main types of mail: Gmail, Hotmail, Yahoo. At the main screen of the Gmail page. You see its parameters.

  1. If you check the "Use Proxy" function, you can open the Proxy.
  2. Open Data: Includes buttons, open mail files, open proxy and open file storage folder
  3. Information: Parameters, input data and check tool process
  4. Start App : Functions to start Tool.
  5. To start the process press the START button. When finished, press STOP.
  6. You can check mail volume up to 30 million mails.










 

At the main screen "Mail Other Mail" is the same, only when checking you need to choose the correct Yahoo and Hotmail format and remember to tick 1 of 2 mail types and start checking.
